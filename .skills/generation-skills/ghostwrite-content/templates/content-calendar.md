# Content Calendar Template

This template defines the format for `CONTENT_CALENDAR_FINAL.md` output.

## Client Defaults (from active_client.md)

- **Client**: {CLIENT_NAME}
- **Founder**: {resolved from founders subcollection}
- **CLIENT_ID**: {CLIENT_ID}

## Variables

| Variable | Description |
|----------|-------------|
| `{PLATFORM}` | Target platform (linkedin, twitter) |
| `{DATE}` | Generation date (YYYY-MM-DD) |
| `{POST_COUNT}` | Total posts generated |
| `{SCHEDULE}` | Posting schedule (e.g., "Mon/Wed/Fri at 9:00 AM ET") |
| `{START_DATE}` | First scheduled post date |
| `{END_DATE}` | Last scheduled post date |
