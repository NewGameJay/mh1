# MH1 Web UI

Marketing Intelligence dashboard for task visibility and content management.

## Setup

```bash
cd ui
npm install
npm run dev
```

Open http://localhost:3000

## Features

- Task dashboard with real-time status
- Content library with approval workflow
- Client selector
- Basic analytics

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- Firebase Firestore
