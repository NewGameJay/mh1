## Usage

### View Module State
```python
from lib.module_manager import get_module

module = get_module("{{MODULE_ID}}")
print(f"Status: {module.status.value}")
print(f"Last updated: {module.updated_at}")
```

### Transition Status
```python
from lib.module_manager import ModuleStatus

# Submit for review
module.transition_to(ModuleStatus.PENDING_REVIEW)
module.save()

# Approve
module.transition_to(ModuleStatus.APPROVED)
module.save()
```

### Start Execution
```python
# Define skill plan
skills = ["skill-1", "skill-2", "skill-3"]

# Start execution
execution = module.start_execution(skills)
print(f"Run ID: {execution.run_id}")
```

### Resume Interrupted Execution
```python
from lib.module_manager import resume_interrupted

# Find interrupted modules
interrupted = resume_interrupted()

for module in interrupted:
    resume_index = module.continue_from_checkpoint()
    if resume_index is not None:
        print(f"Resume {module.module_id} from skill {resume_index}")
```
